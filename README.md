# Netacademia Electron tanfolyam

https://netacademia.hu/Tanfolyam/2017electron-electron

Egy eszköztárral lefedni több platformot mindig is egy nagy elõny volt mind üzleti, mind szakmai szempontból. Electron segítségével képessé válunk desktop alkalmazások fejlesztésére az eddigi angularos tudásunk felhasználásával.
A kurzus során egy alapvetõ admin felületet fogunk megvalósítani az eddigi Eseményvadász rendszerünkhöz, ami utána desktop alkalmazásként lesz futtatható.
A kurzus során az Angular material-al fogjuk felépíteni a felületünket, ngrx/store-t fogjuk használni az Observable alapú állapot kezelésre, használni fogjuk a Redux Devtools Extensiont idõutazásra, kezelni fogjuk a többnyelvûséget és alkalmazni fogunk egy robosztus logolást és hibakezelést.
